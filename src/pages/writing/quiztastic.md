# Quiztastic

Quiztastic is an online multiplayer trivia game. You create a game by choosing a
category, difficulty, and how many lives each player starts with. You are then
given a game code that someone else can use to join your game.

Once in game, the questions are a mix of multiple choice and true/false. The
questions are sourced from [OpenTB](https://opentdb.com/).
