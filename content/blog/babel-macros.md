Babel macros are a way to apply code transformations without having to install a
new plugin for each transformation. They are implemented in a clever way and can
be very useful. And best of all, the work out of the box with Create React App,
GatsbyJS, and NextJS. This post will dive into what they are, how to use them,
and how to make them using TypeScript.
